# Diabetes Prediction Using Machine Learning

This project builds a **diabetes prediction model** using medical diagnostic measurements such as glucose level, BMI, age, insulin, etc.  
It applies machine learning techniques to classify whether a patient is likely to have diabetes.

- Exploratory Data Analysis (EDA)
- Handling missing and skewed data
- Feature scaling (StandardScaler)
